📌 Project Overview

I developed an interactive Bank Marketing Analysis Dashboard using Tableau to reveal insights from bank marketing data.

This dashboard helps businesses and analysts understand customer responses, campaign performance, and trends in marketing outcomes. It enables better decision‑making on targeting, strategy optimization, and identifying patterns that impact conversions.

🔹 Key Features

Interactive Filters: Explore data by customer segments, campaign attributes, and time periods

Performance Metrics: Visualizations showing response rates, campaign effectiveness, and key trends

Actionable Insights: Helps identify high‑impact strategies and areas of improvement

💡 Skills Gained

This project strengthened my expertise in:

Data visualization and analytical storytelling with Tableau

Building interactive dashboards for data interpretation

Extracting insights from complex datasets for business decisions

🛠 Tools & Technologies

Tableau Desktop: Visualization and dashboard development

Excel / CSV: Data preparation and cleaning
